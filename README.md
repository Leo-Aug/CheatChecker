# CheatChecker
一个Python实现的抄袭检测器


## 环境配置

1. 将项目克隆到本地

    ```bash
    git clone https://github.com/Leo-Aug/CheatChecker.git
    ```
2. 安装依赖

    ```bash
    pip install -r requirements.txt
    ```

## 运行程序
1. webAPI形式

    ```bash
    python api.py
    ```
2. 命令行形式

    修改配置文件 config.ini

    项目初始化
    ```bash
    python databaseini.py
    ```

    检测抄袭
    ```bash
    python check.py
    ```
